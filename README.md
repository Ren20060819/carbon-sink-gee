# carbon-sink-gee
Estimating vegetation carbon sink using Sentinel-2 and Google Earth Engine
# 慧眼识碳：基于 Sentinel-2 的区域植被碳汇估算（GEE）

## 项目简介
本项目基于 Google Earth Engine 平台，利用 Sentinel-2 表面反射率数据，
构建“NDVI–植被覆盖度–地上生物量–植被碳储量”的遥感估算流程，
用于评估研究区 2019–2025 年间植被碳汇变化情况。

项目面向水土保持与荒漠化防治背景下的生态恢复效果评估，
强调方法的可复现性与假设透明性。

---

## 研究区
研究区位于黄土高原典型生态脆弱区（ROI 多边形见代码），
以草地/灌丛植被类型为主。

---

## 数据来源
- Sentinel-2 Surface Reflectance（COPERNICUS/S2_SR）
- 时间范围：2019 年生长季、2025 年生长季
- 空间分辨率：10 m

---

## 方法流程
1. Sentinel-2 数据筛选与云掩膜（SCL）
2. NDVI 计算
3. NDVI → 植被覆盖度（FVC）
4. FVC → 地上生物量（AGB）
5. 生物量 → 植被碳储量（Carbon）
6. 不同时期碳储量差值分析（ΔCarbon）

---

## 关键假设与适用范围
- FVC 与地上生物量呈线性关系
- 干生物量中碳含量采用 IPCC 推荐系数（0.45）
- 方法适用于草地与灌丛为主区域，不适用于密闭森林

---

## 已知问题与不确定性
- 不同年份影像获取时间存在差异，可能引入 NDVI 偏差
- AGB 转换系数采用经验值，未进行地面样点校准
- 当前结果主要用于相对变化分析，而非绝对碳储量核算
